Authentication

Token-based login to SAFE

The first thing to know how as a SAFE App Developer is Authentication. This allows your App access to your secure data on SAFE. Try the following example to see it in action!

Javascript Code:

function authorize() {
const app =
    {
    name: "SUCCESS!!!!!!!",
    id: "id",
    version: "0.0.0.0.0.0.0.0.1",
    vendor: "SUCCESSFULMAN!!!",
    permissions: ["SAFE_DRIVE_ACCESS"]
    };
window.safeAuth.authorise( app );
    };

HTML Code:

< button onClick="authorize()">Click to Authenticate!!< /button>
